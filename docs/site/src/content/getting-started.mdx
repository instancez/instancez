## Prerequisites

- **Go 1.25+** (for building from source) or download a prebuilt binary
- **PostgreSQL 15+** — Ultrabase needs a running Postgres instance

## Installation

```bash
go install github.com/ultrabase/ultrabase@latest
```

Or download the binary from the [releases page](https://github.com/ultrabase/ultrabase/releases).

## Create your config

Create an `ultrabase.yaml` file in your project root:

```yaml
project:
  name: my-app
  description: My first Ultrabase project

server:
  port: 8080
  cors:
    origins: ["*"]
    methods: [GET, POST, PATCH, DELETE]

tables:
  todos:
    fields:
      - name: id
        type: bigserial
        primary_key: true
      - name: title
        type: text
        required: true
      - name: done
        type: boolean
        default: "false"
      - name: created_at
        type: timestamptz
        default: "now()"
```

## Start the server

Set the two Postgres login URLs and start the dev server:

```bash
# Provision once on your Postgres server (as a superuser):
#   CREATE ROLE ultrabase_owner LOGIN PASSWORD '...'
#       CREATEROLE CREATEDB BYPASSRLS REPLICATION;
#   CREATE ROLE authenticator LOGIN PASSWORD '...' INHERIT;
#   ALTER DATABASE mydb OWNER TO ultrabase_owner;

export ULTRABASE_OWNER_DATABASE_URL="postgres://ultrabase_owner:pass@localhost:5432/mydb?sslmode=disable"
export ULTRABASE_AUTH_DATABASE_URL="postgres://authenticator:pass@localhost:5432/mydb?sslmode=disable"
ultrabase dev
```

<Callout variant="info" title="Dev mode">
  `ultrabase dev` watches your YAML file for changes and automatically applies migrations.
  For production, use `ultrabase serve` which skips file watching.
</Callout>

## Try it out

Your API is now live. Create a todo:

```bash
curl -X POST http://localhost:8080/rest/v1/todos \
  -H "Content-Type: application/json" \
  -d '{"title": "Learn Ultrabase"}'
```

List all todos:

```bash
curl http://localhost:8080/rest/v1/todos
```

Filter and sort:

```bash
# Incomplete todos, newest first
curl "http://localhost:8080/rest/v1/todos?done=eq.false&order=created_at.desc"
```
