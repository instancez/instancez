The `inz` CLI is a single binary — no runtime dependencies, no package manager required.

<Aside type="tip" title="Code functions require Node.js">
If you plan to use [code functions](/instancez/build/functions/) (`functions:` block in `instancez.yaml`), you'll need **Node.js 18+** and **npm** installed. Everything else — auth, database, storage, RLS — works without them.

Install Node.js (which includes npm) from [nodejs.org](https://nodejs.org) or via your system package manager.
</Aside>

## One-line install

<Tabs>
  <TabItem label="macOS">
  ```bash
  curl -fsSL https://get.instancez.io | sh
  ```
  Installs `inz` to `~/.local/bin`. If that directory isn't in your `PATH`, the script will tell you.
  </TabItem>

  <TabItem label="Linux">
  ```bash
  curl -fsSL https://get.instancez.io | sh
  ```
  Installs `inz` to `~/.local/bin`. Works on x86-64 and ARM64.
  </TabItem>

  <TabItem label="Windows">
  ```powershell
  irm https://get.instancez.io/windows | iex
  ```
  Installs `inz.exe` to `%LOCALAPPDATA%\instancez\bin`. Run in PowerShell 5.1+.

  Alternatively, [download the `.exe` directly](#manual-download) and place it anywhere on your `PATH`.
  </TabItem>
</Tabs>

Verify the install:

```bash
inz --version
```

## Manual download

Download a pre-built binary from [GitHub Releases](https://github.com/instancez/instancez/releases/latest):

| Platform | File |
|---|---|
| macOS (Apple Silicon) | `inz_VERSION_darwin_arm64` |
| macOS (Intel) | `inz_VERSION_darwin_amd64` |
| Linux x86-64 | `inz_VERSION_linux_amd64` |
| Linux ARM64 | `inz_VERSION_linux_arm64` |
| Windows x86-64 | `inz_VERSION_windows_amd64.exe` |
| Windows ARM64 | `inz_VERSION_windows_arm64.exe` |

Make the binary executable (macOS/Linux), rename it, and move it to a directory on your `PATH`:

```bash
chmod +x inz_VERSION_darwin_arm64
mv inz_VERSION_darwin_arm64 /usr/local/bin/inz
```

## Updating

Re-run the install command to update to the latest release. The script overwrites the existing binary.

## Uninstalling

Delete the binary:

<Tabs>
  <TabItem label="macOS / Linux">
  ```bash
  rm ~/.local/bin/inz
  ```
  </TabItem>
  <TabItem label="Windows">
  ```powershell
  Remove-Item "$env:LOCALAPPDATA\instancez\bin\inz.exe"
  ```
  </TabItem>
</Tabs>
